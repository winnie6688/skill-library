# WeChat Official Account Skill Group V1-alpha

这一组 Skills 服务于「要AI不释手」微信公众号内容生产流程。

## Skill list

| Skill | 中文名 | 主要用途 | 当前版本 |
|---|---|---|---|
| `wechat-brand-context` | 微信公众号品牌上下文 | 保存账号定位、栏目体系、视觉风格、内容边界和 GEO 目标 | V1-alpha |
| `wechat-content-polisher` | 公众号正文润色器 | 优化正文逻辑、结构、段落节奏、真实表达和非技术读者理解度 | V1-alpha |
| `wechat-title-generator` | 公众号标题生成器 | 生成清晰、可搜索、不标题党的公众号标题 | V1-alpha |
| `wechat-html-css-layout` | 公众号排版器 | 生成符合品牌色体系的 HTML/CSS 排版 | V1-alpha |
| `wechat-cover-generator` | 公众号封面提示词生成器 | 根据文章主题生成品牌一致的封面方案和图片提示词 | V1-alpha |
| `wechat-geo-summary` | 公众号 GEO 摘要器 | 生成 120 字以内摘要、关键词、长尾问题和 AI 可引用句 | V1-alpha |
| `wechat-fact-checker` | 事实校验器 | 校验技术概念、工具能力、平台规则、经济金融事实等表述 | V1-alpha |
| `wechat-publish-checker` | 发布前检查器 | 对标题、正文、摘要、排版、封面、事实校验和品牌适配做总复核 | V1-alpha |
